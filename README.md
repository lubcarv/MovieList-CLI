#  MovieList CLI

A Node.js command-line application for managing movies. Users can add movie titles, descriptions, and genres, and view organized movie lists through an interactive terminal interface.

---

## Features

* Add new movies with title, description, and genre
* View movies filtered by genre
* Interactive menus via terminal
* Simple and intuitive CLI experience

---

## Technologies

* **Node.js** `v22.21.1`
* **JavaScript (ES6+)**

---

##  Dependencies and Versions

The following dependencies are used in this project:

* **chalk** `4.0.0` – for styling and coloring terminal output
* **inquirer** `8.2.6` – for interactive command-line prompts

---

##  How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/lubcarv/MovieList-CLI.git
```

2. Navigate to the project folder:

```bash
cd MovieList-CLI
```

3. Install dependencies:

```bash
npm install
```

4. Run the application:

```bash
node index.js
```
